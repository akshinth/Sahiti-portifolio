Linear Regression Project

Tools:

Pandas, Kaggle Notebook Env, Machine Learning Algorithms,matplotlib,seaborn


This project is about Ecommerce company based in New York City that sells clothing online but they also have in-store style and clothing advice sessions. Customers come in to the store, have sessions/meetings with a personal stylist, then they can go home and order either on a mobile app or website for the clothes they want.

Objective of the project is to build an ML model which predicts whether to focus their efforts on their mobile app experience or their website.

Loan Repayment Prediction Project

Tools:

EDA, Decision Tree and Random Forest classifiers,Model Evaluation,Scikit-learn


Loan Repayment Prediction Project

This project explores publicly available data from LendingClub.com to predict whether or not a borrower will repay their loan in full. Lending Club connects borrowers with investors, and the goal is to create a model that predicts loan repayment based on borrower profiles.

After basic EDA, Random Forest Regressor model from Scikit-Learn Library was built to predict diagnosis.

K Means Clustering Project: University Classification

Tools:

EDA,K-Means clustering Machine learning algorithem,Model Evaluation,pandas,matplotlib


This Project aims to categorize universities into two groups—private and public—based on various institutional factors. Using machine learning techniques, specifically the KMeans clustering algorithm, the project clusters universities without prior classification into these distinct groups. By evaluating the clustering results against known labels (Private/No), the project assesses the algorithm's effectiveness in differentiating between types of universities based on their characteristics.
